# swaglyrics-bot
GCI project. Fetches lyrics of song that you're currently listening on Spotify via Discord.

## dependencies
- discord.py
- swaglyrics

## how to make it work

 - put your token in the config.py
 - run the main.py
 - send "s!lyrics" to the chat
 - woohoo, you did it
